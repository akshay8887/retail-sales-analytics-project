# Retail Sales Analytics Dashboard

This project analyzes retail sales data to identify product performance, customer behavior, and revenue trends.


## Tools Used

- SQL
- Python
- Power BI

## Project Architecture

Python → Data Generation  
MySQL → Data Storage  
SQL → Data Analysis  
Power BI → Data Visualization


## Project Workflow

Python → MySQL → SQL Analysis → Power BI Dashboard

## Key Insights

## Key Insights

• Phones and laptops generated the highest revenue among all products.

• A small percentage of customers contributed a large share of total revenue.

• Sales trends show steady growth across the analysis period.

• Product revenue distribution highlights the importance of high-value electronics.
